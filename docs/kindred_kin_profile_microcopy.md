# Kin Profile — Microcopy

This defines the exact language for a single Kin (person) view.

The goal:
- No pressure
- No instruction
- Clear separation between *them* and *your notes*
- Everything feels like it’s simply there

---

## 6. Subtle Guidance Patterns

Never tell them what to do.

Use indirect phrasing:

Instead of:
> Add notes about them

Use:
> Things you want to remember.

---

Instead of:
> Track important dates

Use:
> Dates you keep

---

## 7. What This Screen Should Feel Like

- Quiet room
- Nothing asking anything from you
- Everything already allowed

---

## 8. Final Check

Before shipping:

- Does anything feel like a task?
- Does anything sound like instruction?
- Does anything create urgency?

If yes → remove it

---

## Final Principle

This screen does not ask you to act.

It simply holds the person in front of you.
